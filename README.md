# E-Commerce Sales Analysis

## Project Overview

This project demonstrates an End-to-End E-Commerce Sales Analysis using Microsoft Excel, MySQL, and Power BI. The project includes data cleaning, SQL-based business analysis, and interactive dashboard development to generate meaningful business insights.


## Tools & Technologies Used

- Microsoft Excel (Data Cleaning & Preprocessing)
- MySQL (Data Analysis)
- Power BI (Data Visualization & Dashboarding)


## Repository Contents

- CSV Files (Cleaned Dataset)
- MySQL SQL Queries
- Power BI Project (.pbix)
- Sales Dashboard Screenshot
- Customer Dashboard Screenshot
- Product Dashboard Screenshot


## Power BI Dashboards

### Dashboard 1 – Sales Performance Analysis
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Monthly Revenue Trend
- Revenue by State
- Top 10 Customers by Revenue

### Dashboard 2 – Customer & State Analysis
- Customers by State
- Orders by State
- Top 10 Customers by City
- Customer Distribution by State
- Orders vs Customers by State

### Dashboard 3 – Product & Payment Analysis
- Revenue by Product Category
- Orders by Product Category
- Revenue vs Orders by Product Category
- Revenue by Payment Type
- Orders by Payment Type
